Privacy Policy

How we handle data on the Apache Unomi website.

Information We Collect

The Apache Unomi website uses Apache Matomo for privacy-respecting analytics. We have configured Matomo with:

  • DoNotTrack is honored — if your browser sends a DNT header, no analytics data is collected
  • No cookies are set by analytics
  • IP addresses are anonymized

Our web server automatically records standard access log information including:

  • Anonymized IP address
  • Date and time of request
  • Pages visited
  • Browser type and operating system
  • Referring URL

How We Use This Information

This information is used solely to improve the Apache Unomi website and understand how visitors use it. It is never sold, shared, or used for advertising.

Third-Party Services

This website loads resources from:

  • Google Fonts (Inter typeface)
  • jsDelivr CDN (Bootstrap, Mermaid)

These services may collect standard HTTP request data as described in their respective privacy policies.

Contact

For questions about this policy, contact the Apache Unomi community via the mailing lists.
