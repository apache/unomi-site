How to Contribute

There are many ways to contribute, no matter your experience level:

Answer Questions

Help users on the mailing list or Slack.

Improve Documentation

Fix typos, add examples, or write new guides.

Report Bugs

File issues on JIRA.

Submit Code

Send pull requests with bug fixes, features, or examples.

Review Designs

Discuss proposals on dev@unomi.apache.org.

Spread the Word

Write blog posts, give talks, or share your experience.
