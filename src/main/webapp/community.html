Contact Us

There are many ways to reach the Apache Unomi user and developer communities - use whichever one seems best!

How to contact us When to use it
users@ mailing list Users discussions (Subscribe, Unsubscribe)
dev@ mailing list Development discussions (Subscribe, Unsubscribe)
commits@ mailing list Firehose of commits, bugs, pull requests, etc. (Subscribe, Unsubscribe)
JIRA bug tracker Report bugs / discover known issues
Slack Report bugs / discover known issues

If you wish to report a security vulnerability, please contact security@apache.org. Apache Unomi follows the typical Apache vulnerability handling process.
