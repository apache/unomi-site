Articles & Slides

  • Customizing the User Experience with Open Source Tech

    Explains what a CDP is and why you don’t need to be Google or Facebook to personalize content. Introduces Unomi’s modular OSGi architecture, rules, segments, and how it compares to proprietary alternatives. — Otávio Machado, Medium, 2018

  • Unomi Out of the Box

    Quick-start guide showing how to bootstrap Unomi from a binary distribution or source code, validate it’s running, and links to the rest of the tutorial series. — Otávio Machado, Medium, 2018

  • Giving Unomi Intelligence

    Deep dive into Unomi’s rule engine: how rules, conditions, actions, and segments work together. Includes Java code examples of built-in evaluators and how to create custom condition and action plugins. — Otávio Machado, Medium, 2018

  • Talking to the Unomi Events API

    Walkthrough of Unomi’s event endpoints (context.js, context.json, eventcollector). Shows request/response formats for sending user events and retrieving profile data with segments. — Otávio Machado, Medium, 2018

  • Unomi: A Bridge Between Privacy and Digital Marketing

    Interview-driven article on Unomi’s role in balancing user privacy with personalization. Covers the OASIS Context Server standard, Unomi’s acceptance into the Apache Incubator, and the growing need for ethical data handling. — Linux.com

  • Introducing Apache Unomi — JavaOne 2015

    47 slides covering Unomi’s architecture, privacy features, and the OASIS Context Server standard. Features a live IoT demo integrating beacons, an iOS app, and Apple TV digital signage with Unomi. — Serge Huber, SlideShare

  • Apache Unomi In-Depth — ApacheCon EU 2015

    43-slide technical deep dive into Unomi’s Karaf-based architecture, services (profiles, privacy, queries, segments), event flow, rules engine, and plugin system. — Serge Huber, SlideShare
