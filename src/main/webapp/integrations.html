Recommended

Groovy Actions

The easiest way to extend Unomi. Write custom logic in Groovy scripts that are hot-deployable — no compilation, no restart needed. Groovy actions can react to events, update profiles, call external APIs, and integrate with third-party systems like CRMs, email platforms, or any REST service.

Start here for most integration needs.

Groovy actions guide Source code

Custom Java Plugins (OSGi)

For advanced needs that go beyond what Groovy actions can do — such as custom conditions, new persistence types, or deep integration with the Unomi core — build a full OSGi plugin in Java. This gives you access to the entire Unomi API and service layer.

Use this only when Groovy actions are not sufficient.
